简单的语法高亮widget

2012-4-16

现在开始写日志。按原来的计划，第一篇日志应该是关于quine程序的，那个小东西前不久曾短暂地迷住了我。其实不管写什么，基本上都需要放上几行代码，才显得专业。要显示代码，只用<code>显然不够，需要语法高亮，还要有整齐的缩进。有不少优秀的语法高亮js代码，但既然自己开博客了，这样的小工具，不正是练手的好机会么。我写了一个配合jQuery使用的simple-highlight.js，配套有simple-highlight.css，用于基本的布局。

下面是一个样例。在页面中适当的位置创建一个用来显示代码的div，给这个div一个合适的大小，并按自己的风格写一段代码，然后这样：$(thisDiv).simpleHighlight()，语法高亮的代码就呈现出来了。

/* This is hello world */ #include <stdio.h> int main(void) { /* * hello world may be easy, * but can you say hello world twenty times? */ int i; float f = 0.8; // just try to highlight a float number. for(i = 0; i < 9; i++) { printf("hello world\n"); } if(i == 10) { while(i < 20) { printf("hello world\n"); i++; } } else printf("you can never print me\n"); return 0; }

我使用javascript时间不长，simple-highlight.js写得比较繁琐，特别是没有使用正则表达式，不知道如果用了的话，代码会清晰多少。这段代码可支持高亮C语言和大部分javascript的关键字，java的主要关键字，支持块注释和行注释，高亮数字，不支持字符串换行，不支持html的各类标记。特别遗憾的是，在IE里完全不行……

如果你对simple-highlight.js感兴趣，可前往github/xie-yu处下载。欢迎回馈bug。
